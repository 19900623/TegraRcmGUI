# TegraRcmGUI
A simple C++ GUI for [TegraRcmSmash](https://github.com/rajkosto/TegraRcmSmash) by [rajkosto](https://github.com/rajkosto)
- Inject payloads
- Run Linux on your switch ([ShofEL2](https://github.com/fail0verflow/shofel2))
